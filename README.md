# EduOS – Premium Digital Education Platform

EduOS is a Harvard-level digital education operating system built in Abuja, Nigeria, for students worldwide.

### Features
- Personalized AI-powered learning paths
- Programs: Primary, Secondary, Military (Navy, Airforce, Intelligence), Sports, Arts/Music, World Religions
- AI Copilot for homework help, leadership coaching, discipline guidance
- Parent oversight dashboard
- Certificates & badges
- Secure payments (Stripe + Google Pay, Paystack for Nigeria)

### Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Firebase (auth, database), Stripe/Paystack (payments)
- Hosting: GitHub Pages (static) + Render.com (backend)

### Live Site
https://victorbosah50-code.github.io/aegis-academy-/

### Getting Started
1. Clone or fork this repo
2. Add your Firebase config to script.js
3. Add Stripe/Paystack keys
4. Deploy backend to Render.com
5. Enjoy!
