# EduOS – Premium Digital Education Platform

EduOS is a modern, AI-powered learning platform built in Abuja, Nigeria, for kids, teens, and adults worldwide.

## Features
- Personalized AI copilot for homework, leadership, discipline, religions
- Programs: Primary, Secondary, Military (Navy, Airforce, Intelligence), Sports, Music, World Religions
- Parent dashboard for progress & certificates
- Modern, educational design with fun videos
- Secure payments (Stripe + Google Pay, Paystack for Nigeria)
