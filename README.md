# EduOS – Premium Digital Education Platform

EduOS is a Harvard-level digital education system built in Abuja, NG, for kids, teens, and all ages worldwide.

## Features
- AI copilot for homework, leadership coaching, discipline
- Programs: Primary, Secondary, Military (Navy, Airforce, Intelligence), Sports, Music, World Religions
- Parent dashboard for progress and reports
- Certificates on completion
- Modern, educational design with fun videos for kids
- Secure payments (Stripe/Google Pay, Paystack for Nigeria)

https://victorbosah50-code.github.io/aegis-academy-/

2. Add keys for Stripe, Paystack, xAI
